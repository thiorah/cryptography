# Secure File Encryption and Authentication System

## Project Description

This project demonstrates practical cryptographic implementations using Python. The system includes AES encryption, RSA encryption, password hashing, and secure authentication concepts for cybersecurity learning and secure communication workflows.

---

## Technologies Used

* Python
* Cryptography Library
* OpenSSL
* VS Code
* GitHub

---

## Features

* AES Symmetric Encryption
* RSA Asymmetric Encryption
* SHA-256 Password Hashing
* Secure Message Encryption
* User Input Validation
* Encryption and Decryption Testing
